## Features
- Beautiful modern UI
- Hero section with quick-write button
- Reading section with trending categories
- Rich blog editor with
- Title & title image
- Text + inline image insertion
- Auto draft saving to localStorage
- Smart text reflow when adding/removing images
- Full CRUD for blogs
- Edit blog with content-image position tracking
- Profile management (goals, interests, avatar)
- Authentication + protected routes
- MyPosts page with clean grid layout
- Mobile-friendly responsive design
